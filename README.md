# AZTEC+

Cleaned version of the AZTEC extension workspace, organized for benchmarking, framework experiments, and plotting utilities.

## Contents

- `Benchmark/`: benchmark experiments and CoNEXT-related scripts
- `Framework/`: core training, allocation, and forecasting code
- `Plotter/`: plotting and analysis notebooks/scripts
- `citys/`: city-level input data
